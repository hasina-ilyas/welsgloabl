## Component rules

- Use deep forest green as the structural base and warm beige for the highest-priority action.
- Keep one dominant CTA per decision area.
- Use rounded controls consistently, with clear focus-visible states.
- Use uppercase eyebrow text sparingly for hierarchy.
- Maintain generous whitespace and readable line lengths.
- Use icons with consistent outline weight.
- Avoid exposing public file URLs in any purchase or delivery state.
- Keep messaging direct, practical, confident, and outcome-oriented.
